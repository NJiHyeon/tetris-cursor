보드와 블록 렌더링 코드를 리뷰해줘.

리뷰 기준:
1. DOM 구조가 명확한가? (게임 보드, 다음 블록, 점수 영역 등이 적절히 분리되어 있는가?)
2. 렌더링 함수가 역할별로 나뉘어 있는가? (보드 그리기, 현재 블록 그리기, 고정 블록 갱신 등)
3. 렌더링 로직이 게임 상태와 분리되어 있는가? (상태 변경과 화면 갱신이 혼재되지 않는가?)
4. CSS와 JS가 중복 없이 협력하는가? (스타일은 CSS, 배치·갱신은 JS)
5. 코드 가독성이 좋은가? (함수명, 변수명이 의도를 드러내는가?)
6. 불필요한 DOM 조작이나 전체 재렌더링이 반복되지 않는가?
